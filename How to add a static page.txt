1) Adding js, css and images to HTML

step1 : create a folder "static" 

static 
  css
    style.css
  js
    script.js
  images

step2 :  link those css and js files in HTML

{% load static %}  : add this line at the top of the HTML

<link type="stylesheet" href="{% static "css/style.css" %}"

<script src="{% static "js/script.js" %}"></script>

<img src="" alt=""/>
